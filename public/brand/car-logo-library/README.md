# GSYEN Car Logo Library

This folder stores curated vintage car logo assets for future brand work.

## Production Source

The current production car logo is:

- `curated/95-round-tire/`

This is the selected `95 round tire` version used for the live GSYEN header and
empty-state car mark.

## Curated Full Marks

Only visually useful candidates are preserved here. Drafts that were merely noisy,
overworked, or redundant were intentionally left out.

- `95-round-tire/` - selected production direction; balanced body and rounder tire.
- `95-soft-round-tire/` - softer tire alternative, useful if the production tire later feels too sharp.
- `95-restrained/` - restrained wheel/body contrast; calmer fallback.
- `burr-cleanup/` - cleaner full-body pass after removing obvious rough burrs.
- `burr-cleanup-small/` - small-size-oriented cleanup; useful for header-scale testing.
- `candidate-08-from-95/` - editable study direction from the 95 base.
- `latest-ink/` - later full-body drawing with stronger editorial silhouette.

Each complete candidate keeps the available variants:

- `ink-transparent.svg`
- `ink-paper.svg`
- `light-transparent.svg`
- `light-dark.svg`

Some candidates only had partial variants when archived.

## Structure

`structure/` contains editable component-level material, not final logo choices.

- `structure/parts/` - the 14 separated car parts.
- `structure/studies/` - layout studies, editable schedule, and separated structure sheets.
- `structure/wheels/` - wheel-only studies for tire/spoke refinement.

Use this folder when rebuilding the mark as a truly editable SVG system.

## Skeleton

`skeleton/` contains modular skeleton drafts:

- `modular-skeleton-v1.svg`
- `modular-skeleton-v2.svg`

These are not final artwork. They are construction guides for future hand-redraw
passes.

## Rejected Asset Types

The following categories were not copied here:

- purely noisy trace experiments
- wheel-only experiments that lost the car's original feeling
- nearly identical intermediate exports
- versions with obvious ray noise or broken wheel balance
